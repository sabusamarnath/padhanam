# Archive

Old material lives here. Nothing is ever deleted.

Archived at audit boundaries:

- `archive/packages/` — closed `current-package.md` snapshots, one per package
- `archive/sessions/` — session log entries from closed phases, moved out of the active log to keep it scannable
- `archive/decisions/` — decisions superseded by later decisions, moved out of the active log with a reference to what replaced them

Archives are read-only after archival. To revisit an archived decision, copy a new entry into the active log with reasoning.
